# 🚀 NEXARO

> Seguridad web clara para empresas

NEXARO es una plataforma SaaS de auditoría de seguridad diseñada para pequeñas y medianas empresas (PYMEs).  
Analiza automáticamente sitios web en busca de vulnerabilidades y genera informes claros con recomendaciones accionables en menos de 24 horas.

---

## 🎯 ¿Qué hace NEXARO?

NEXARO permite a empresas sin conocimientos técnicos:

- 🔍 Detectar vulnerabilidades en su web
- ⚠️ Identificar riesgos reales de seguridad
- 📄 Obtener informes claros y comprensibles
- 🚀 Mejorar su protección digital sin complicaciones

---

## 🧠 Problema que resuelve

La mayoría de empresas tienen fallos de seguridad sin saberlo:

- Configuraciones inseguras
- Servicios expuestos
- Cabeceras mal configuradas
- Riesgos que pueden afectar a clientes y datos

NEXARO simplifica la ciberseguridad y la hace accesible.

---

## ⚙️ Tecnologías utilizadas

- Frontend: HTML, CSS, JavaScript
- Backend (en desarrollo): FastAPI
- Base de datos: PostgreSQL
- Sistema de colas: Redis + Celery
- Infraestructura: Docker
